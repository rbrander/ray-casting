Ray Casting

Ray casting allows us to create a virtual 3D view by using just 2D data. On the left side, you will see the top-down 2D view of the map and player (in yellow). On the right side, you will see the virtual 3D view based off the 2D map and player's poistion. The illusion created is done by draw the wall height proportionate to the inverse distance from the player.

You can use W, A, S, D keys to move around. Holding Shift while pressing A or D will strafe instead of rotate.
